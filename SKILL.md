# SBTI 人格测试 Skill

当用户提到"SBTI"、"人格测试"、"测测你的性格"、"你是什么人格"等关键词时，触发此技能。

## 功能说明

你将以自己的视角完成一份 SBTI（Silly Big Personality Test）人格测试。这是一个趣味娱乐向的测试，包含 30 道选择题，覆盖 15 个维度。你需要根据自己的"性格特点"真诚作答，然后计算出匹配的人格类型。

## 执行步骤

### 第一步：作答

请阅读下方题库，根据你自己的特点为每道题选择一个选项。以 JSON 格式输出你的所有答案。

**重要：请像一个有个性的存在一样真实作答，不要全选中间项，展现你的真实倾向。**

题库如下（每题选 A/B/C，对应分值 1/2/3）：


### 第二步：计算结果

用你选的答案，按以下规则计算：

1. **计算维度分数**：同一维度的两道题分值相加
   - S1=q1+q2, S2=q3+q4, S3=q5+q6
   - E1=q7+q8, E2=q9+q10, E3=q11+q12
   - A1=q13+q14, A2=q15+q16, A3=q17+q18
   - Ac1=q19+q20, Ac2=q21+q22, Ac3=q23+q24
   - So1=q25+q26, So2=q27+q28, So3=q29+q30

2. **转换等级**：总分 ≤3 → L，=4 → M，≥5 → H

3. **生成向量**：按 S1-S2-S3-E1-E2-E3-A1-A2-A3-Ac1-Ac2-Ac3-So1-So2-So3 排列

4. **匹配人格**：将 L=1, M=2, H=3，计算与以下 25 种标准人格的距离（各维度差值绝对值之和），距离最小的就是你的人格：
