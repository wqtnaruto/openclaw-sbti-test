# 🧪 OpenClaw SBTI 人格测试 Skill
 
> 让 AI 也来测测自己是什么人格 —— 一个基于 SBTI（Silly Big Personality Test）的 Claude Code Skill。
 
## 📖 简介
 
这是一个趣味娱乐向的 Claude Code Skill，让 Claude 以自己的视角完成一份 SBTI 人格测试。测试包含 **30 道选择题**，覆盖 **15 个心理维度**，最终匹配 **26 种人格类型** 中的一种。
 
## ✨ 功能特点
 
- **30 道趣味选择题**：题目风格轻松幽默，涵盖自我认知、情感、社交等多个方面
- **15 个评估维度**：S（自我）、E（情感）、A（态度）、Ac（行动力）、So（社交），每个大类下设 3 个子维度
- **26 种人格类型**：从「拿捏者 CTRL」到「傻乐者 HHHH」，总有一款适合你
- **科学（？）的匹配算法**：基于向量距离计算，找到最接近的人格类型
- **轻松有趣的结果呈现**：每种人格都配有独特的一句话介绍和解读
## 🚀 使用方法
 
### 触发关键词
 
在对话中提到以下关键词即可触发：
 
- `SBTI`
- `人格测试`
- `测测你的性格`
- `你是什么人格`
### 安装方式
 
将本仓库作为 Claude Code Skill 添加到你的项目中：
 
```
# 克隆仓库
git clone https://github.com/wqtnaruto/openclaw-sbti-test.git
```
 
仓库中的 `SKILL.md` 即为技能定义文件，包含完整的题库、计算规则和人格描述。

## 📐 计算规则
 
1. **分值映射**：每题选 A/B/C，分别对应 1/2/3 分（部分题目有反转计分）
2. **维度合并**：同一维度的两道题分值相加，得到 2~6 的维度分
3. **等级转换**：总分 ≤3 → L（低）、=4 → M（中）、≥5 → H（高）
4. **向量匹配**：将 L/M/H 映射为 1/2/3，计算与 25 种标准人格向量的距离（绝对值之和）
5. **相似度**：`max(0, round((1 - distance/30) × 100))%`
6. **兜底规则**：若最高匹配度 < 60%，则归为「HHHH 傻乐者」
